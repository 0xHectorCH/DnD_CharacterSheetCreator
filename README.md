# DnD_CharacterSheetCreator
Página de creación, almacenaje de hojas de personaje de DnD y de revisión de reglas. El proyecto es una colaboración por ello no se hará la hoja de personaje aún, pero si el catálogo de reglas.

Los archivos --X-- estan aun en desarrollo o son para uso futuro, evitarlos. Se encontrarán comentarios a lo largo del código relacionados con esto mismo.

Tecnologias utilizadas: React, Tailwindcss, Vite, Typescript, JsonServer, Git
